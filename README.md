## Repository documentation

This project is a web application of a to-do list and was developed with HTML, CSS, JavaScript, JQuery, PHP and SQL. Project created during my Web Development studies.

> My tasks

This project is a to-do list where the user can create, read, update or delete a task.


<img src="/img/readme-image-1.png">

> Database

The database consists of a table called task with the columns id and description. A text file with the SQL commands used can be found [here](/database). To connect to the database, you need to create the "connection-string.php" file inside the "database" directory, passing your connection settings.
